ISLAMABAD EMERGENCY RESPONSE SYSTEM
====================================

A real-time emergency management system for Islamabad, Pakistan.

SYSTEM FEATURES:
----------------
1. Real-time monitoring of all Islamabad sectors
2. Emergency simulation (earthquake, fire, flood, medical)
3. Multi-agency coordination (Rescue 1122, CDA, Police, Hospital)
4. Mass SMS alert system simulation
5. Citizen reporting system
6. Real-time weather integration
7. Live event logging and reporting
8. Government presentation generator


KEYBOARD SHORTCUTS:
-------------------
Ctrl+E : Simulate Earthquake
Ctrl+F : Simulate Fire
Ctrl+S : Send Mass SMS Alert
Ctrl+P : Generate Presentation

FOR GOVERNMENT PRESENTATION:
---------------------------
1. Run the system
2. Simulate some emergencies
3. Click "Generate Presentation" button
4. Present the exported file to authorities
